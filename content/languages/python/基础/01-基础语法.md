# 第一部分：Python 基础核心知识点
## 1. 模块导入 (import)
### 知识点定义
导入Python内置模块/第三方库，使用库中的函数、类、变量。
### 代码示例
```python
import feedparser
import time
import os
from datetime import datetime
```
### 学习内容
- 标准导入：`import 模块名`
- 从模块导入指定内容：`from 模块 import 类/函数`
- 别名导入：`import 模块 as 别名`
- 内置模块 vs 第三方模块

## 2. 变量与数据类型
### 知识点定义
存储数据的容器，Python常用数据类型。
### 代码示例
```python
result = {"result": []}  # 字典
title = entrie["title"]  # 字符串
feed_entries_length = len(feed_entries)  # 整数
```
### 学习内容
- 字符串(str)、整数(int)、列表(list)、字典(dict)
- 类型判断：`isinstance(tmp_rss_info_atom, int)`
- 变量赋值、修改、拼接

## 3. 条件判断 (if/else/elif)
### 知识点定义
根据条件执行不同代码逻辑。
### 代码示例
```python
if(request_success == False):
    pass
else:
    pass
if(len(rss_info) > 0):
    pass
```
### 学习内容
- 条件表达式：==、>、<、>=、<=、!=
- 多分支判断：if-elif-else
- 逻辑运算符：and/or/not

## 4. 循环语句 (for/while)
### 知识点定义
重复执行一段代码。
### 代码示例
```python
for i in range(3):  # 固定次数循环
    pass
for entrie in feed_entries[:]:  # 遍历列表
    pass
```
### 学习内容
- `for ... in ...` 遍历列表/字典/字符串
- `range()` 生成数字序列
- `while` 条件循环
- 循环控制：break/continue

## 5. 函数定义 (def)
### 知识点定义
封装可重复使用的代码块。
### 代码示例
```python
def get_rss_info(feed_url, index, rss_info_list):
    pass
```
### 学习内容
- 定义语法：`def 函数名(参数):`
- 形参、实参
- 返回值：`return`
- 函数调用

## 6. 异常处理 (try/except)
### 知识点定义
捕获程序运行错误，避免崩溃。
### 代码示例
```python
try:
    feed_url_content = requests.get(...)
except Exception as e:
    print("出错",e)
```
### 学习内容
- `try-except` 捕获异常
- `Exception` 通用异常
- 异常信息打印
- 容错机制

## 7. 字符串处理
### 知识点定义
对文本进行替换、清洗、拼接。
### 代码示例
```python
title = title.replace("\n", "")  # 替换字符
str(i)  # 类型转换
```
### 学习内容
- `replace()` 替换
- `strip()` 去空格
- 字符串拼接
- 类型转换：str()/int()

## 8. 内置函数
### 知识点定义
Python自带的工具函数。
### 代码示例
```python
len(feed_entries)  # 长度

```
### 学习内容
- `len()`/`print()`/`range()`
- 类型转换函数
- 常用工具函数

# 第二部分：Python 进阶知识点
## 9. 正则表达式 (re模块)
### 知识点定义
**文本匹配/提取/替换**，脚本核心功能。
### 代码示例
```python
link = re.findall(r'\[订阅地址\]\((.*)\)', before_info)[0]
reResult = re.findall(mail_re, new_edit_readme_md[0])
```
### 学习内容
- `re.findall()` 提取所有匹配内容
- 正则语法：`()`分组、`.*`匹配任意字符
- 原始字符串 `r''`
- 文本清洗、模板替换

## 10. 多进程并发 (multiprocessing)
### 知识点定义
**同时爬取多个RSS**，大幅提升速度。
### 代码示例
```python
from multiprocessing import Pool, Manager
po = Pool(8)  # 进程池
po.apply_async(get_rss_info,(link, index, rss_info_list))
po.close()
po.join()
```
### 学习内容
- 进程池 `Pool`
- 异步任务 `apply_async`
- 进程共享数据 `Manager().list()`
- 并发、等待子进程完成

## 11. 日期时间处理 (datetime/time/pytz)
### 知识点定义
获取时间、格式化、时区转换。
### 代码示例
```python
datetime.today().strftime("%Y-%m-%d")
datetime.fromtimestamp(...,pytz.timezone('Asia/Shanghai'))
```
### 学习内容
- 获取当前时间
- 时间格式化 `strftime`
- 时区处理 `pytz`
- 时间戳转换

## 12. 文件操作 (os/shutil)

读写文件、复制文件夹、判断路径。
### 代码示例

```python
with open("EditREADME.md",'r') as load_f:

shutil.copyfile(...)
shutil.copytree(...)
os.path.exists(...)
```
### 学习内容
- `with open()` 安全读写文件
- `os` 路径/文件判断
- `shutil` 复制/删除文件/文件夹
- 路径拼接 `os.path.join()`
